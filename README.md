# 🏀 2023-2024 NBA Player Stats Dashboard

## 📝 Project Description
This project analyzes a comprehensive dataset of the 2023-2024 NBA season, covering performance metrics for both the Regular Season and the Playoffs. The dataset includes statistics for 572 unique players across 30 teams, with player ages ranging from 19 to 39 years old.

The main goal is to process the data and create an interactive visualization that helps explore player efficiency, scoring distributions, and team performance trends.

---

## 🛠 Tools Used
- Power BI → Visualization and interactive dashboard design

---

## 🚀 Workflow
1. Collection: Gathering and preparing the 2023-2024 NBA season dataset.
2. Cleaning: Handling missing values, duplicates, and inconsistencies to ensure data accuracy.
3. Normalization: Standardizing and structuring the dataset to better clarify relationships between player stats and team performance.
4. Visualization: Building an interactive Power BI dashboard that highlights key KPIs and performance metrics.
